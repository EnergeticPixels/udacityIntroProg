Stage 0 Notes

HTML with <b>, <p>, and <em> tags

  • the bold tag is designated for stressing textual (visual) elements on the screen. But recent standards advise to use the strong tag and then style that tag in css to get the desired visual result.
  • the paragraph tag is for designating paragraphs of text. The visual result is something like printed material, but in our case it comes along with some 'automatic' styling for spacing between other elements around the paragraph.
  • The emphasis tag is for styling something that should be in italics or to give a textual element some other 'embossing'

HTML line-break tag

  • The html element for line break is probably thought of as being a block element, but I have learned that it is thought of as being an inline element. As crazy as that sounds.
